# FreeAISync
FreeAISync 是一个面向 macOS / Windows 的免费开源文件夹同步工具，专为手机、U 盘、移动硬盘、照片库、视频库和大型文件资料管理而设计。  它不仅仅是一个传统的文件同步软件，更是一个支持自然语言交互的 AI 资料管理助手。你可以像和助理对话一样告诉它：  “把我手机里的照片按年月同步到移动硬盘。” “只同步 4K 视频，不要重复文件。” “帮我把这个 U 盘和备份盘做一次双向同步，但跳过临时文件。” “找出哪些大文件占用了最多空间，并给我整理建议。”  FreeAISync 目标是让复杂的文件同步、照片归档、视频整理、空间管理和多设备资料备份变得简单、透明、可控。
